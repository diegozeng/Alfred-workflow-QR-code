
##Alfred workflows QR code (v1.0)
---------------
####好吧，我又无聊了.最近经常要把电脑上看到的文章转到ipad上阅读，所以就写了一个简单的alfred workflow，自动生成输入网页的二维码，之后的步骤“扫一扫”大家都懂的。思路很简单: 直接调用联图网提供的的api。其实谷歌浏览器已经有不错的插件可以实现了，不过懒人还在坚持用自带的safari。
